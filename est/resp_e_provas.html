(X) A ∩(B ∪ C) = {0, 1, 2, 3, 4}

Resolução:

Para resolver essa questão precisamos primeiro entender o que cada evento representa.
A = {n < 5} = {0, 1, 2, 3, 4}
B ∪ ¬ C = {3, 4, 5, 6, 7} ∪ ¬ {4, 6, 8}
= {3, 4, 5, 6, 7} ∪ {0, 1, 2, 5, 7, 9}
= {0, 1, 2, 3, 4, 5, 6, 7, 9}
portanto:
A ∩(B ∪ ¬C) = {0, 1, 2, 3, 4}
