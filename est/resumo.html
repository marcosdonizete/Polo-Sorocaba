Semana 1

Estatística Descritiva

  • Organizar, classificar e tabelar os dados segundo algum critério de maneira a facilitar sua interpretação.
  • Resumir, sintetizar os dados (em geral, volumosos).
  • Apresentar os dados através de figuras com rigor metodológico.

Dados

Informações provenientes de observações, medições ou respostas.

População

Coleção de todos os resultados, repostas ou medições que são de interesse.

Amostra

Um subconjunto ou parte de uma população que deve representar da melhor maneira possível a população à qual ela pertence. O que é população e o que é amostra pode varia conforme o objeto da pesquisa.

Tipos de amostras

  • Amostra aleatória simples (AAS) → amostras de mesmo tamanho possuem mesma probabilidade de serem selecionadas ao acaso. Chamada de AAS com reposição
  • Amostra aleatória estratificada → quando a população possui estratos (faixa etária ou renda) a amostra representa esses estratos proporcionalmente. É uma filtragem na população. Exemplo: pirâmide etária.

Variáveis estatísticas → são os registradores (nomes) usados para armazenar dados de tipo bem definido.

Tipos de Variáveis

  • Numéricas ou Quantitativas → usadas para armazenar dados numéricos. Ex.: massa, volume, tempo, anos de instrução, idade, salário etc. Podem ser de dois tipos:
    • Discretas → resultantes de contagens (em geral valores inteiros) finitas e mensuráveis como número de irmãos, de defeitos etc. Um conjunto discreto e finito de valores (não precisam ser inteiros). Ex.: idade (em anos), pessoas em uma fila, altura em centímetros
    • Contínuas → assumem valores em intervalos de números reais como peso, altura etc. Dados que podem assumir valores em um intervalo real. Ex.: preços, duração de uma música, índice pluviométrico.

    Observações → A classificação depende de certas particularidades. Idade medidas em número de anos pode ser classificada como como discreta, porém, se classificada por números de dias então ela será contínua (2,85 anos). Deve-se utilizar o bom senso na hora de de decidir que procedimento adotar para caracterizar uma variável

  • Não Numéricas ou Qualitativas (categórica) → usadas para armazenar dados que consistem de atributos, rótulos ou entradas não numéricas. Ex.: estado civil, preferência de consumo, grau de instrução, cor dos olhos. Podem ser de dois tipos:
    • Qualitativas ordinais → tem uma ordenação natural indicando intensidades crescentes de realização. registram dados que podem ser ordenados de alguma forma. Ex.: grau de instrução (fundamental, médio, superior), classe social (A, B, C,...), tamanho (P, M, G)
    • Qualitativas nominais → quando não é possível estabelecer uma ordem natural entre seus valores. Classificam os dados em diferentes categorias. Ex.: estado civil, região de procedência, sim/não.Exemplo: M/F, tamanho (P/M/G) etc.

Níveis de mensuração de variáveis

  • Escala nominal → dados categóricos, não se pode realizar cálculos matemáticos com eles.
  • Escala ordinal → os dados podem ser ordenados mas diferenças entre valores não tem sentido matemático. Ex.: quanto é G - M?
  • Escala intervalar → os dados podem ser ordenados e diferenças entre valores têm sentido matemático. Geralmente são números. Ex.: 30°C - 20°C = 10°C.
  • Escala da razão → os dados podem ser ordenados, as diferenças entre valores têm sentido e escala possui um "zero natural". Os dados podem ser expressos com múltiplos uns dos outros. Ex.: família com 0 filhos.

Frequência Absoluta (n)

Lista e quantifica os valores possíveis de uma variável separadamente do conjunto bruto de dados.

Frequência Relativa (fi= ni/n)

Determina o percentual de ocorrência dos valores de uma determinada variável. Exemplo

Gênero ni fi fac
F 37 0,74 0,74
M 13 0,26 1
total n = 50 1 1

Gráficos

  • Setor (pizza) → se adapta muito bem às variáveis qualitativas.
  • Barras → se adapta melhor às variáveis discretas ou qualitativas.
  • Histograma (retângulos contíguos) → onde a altura de cada retângulo é denominada densidade e é definida pelo quociente da área pela amplitude da faixa lateral. Alguns autores utilizam a frequência absoluta ou a porcentagem na construção do histograma. O histograma também pode ser utilizado no cálculo da mediana que o valor que divide o conjunto de dados ordenados em dois subgrupos de mesmo tamanho (das observações ordenadas, 50% estão abaixo e 50% estão acima das medianas) definido por mobs.

Semana 2

Organização e apresentação de dados

  • informação ≠ dados
  • Organizar os dados de forma que seja possível exibir algumas características. Ex.: quantis
  • Agrupar e contar valores semelhantes em classes
  • Apresentar dos dados através de gráficos que ilustrem suas características

Distribuição de frequências

  • permite organizar os dados, extrair medidas e informações dos dados
  • É a partir da tabela de frequências que se constrói o histograma.
  • distribuição empírica de probabilidades, histograma.
  • Para construir uma distribuição de frequências
    • decidir em quantas classes agrupar os dados
      • poucas classes agrupam os dados demais
      • muitas classes agrupam os dados de menos
      • o típico é uma tabela ter de 4 a 20 intervalos de classe
      • para um número pequeno (n ≤ 200) a regra de Sturge sugere → K = [1 + log2n] = [1 +3,322log10n]
      • encontrar uma amplitude de classes conveniente
        • calcular a amplitude dos intervalos de classe (K) por → Δ = (max(x) - min(x)) / K
        • convém usar um valor de delta inteiro caso dê quebrado
      • determinar os limites inferior e superior das classes (cobrindo toda a amplitude)
        • começar a partir de um valor conveniente inicial ≤ min(x)
        • a partir dele se recobrimos a amplitude dos dados com sucessivos intervalos de classe de largura delta.
        • pode ser necessário usar mais intervalos de classe (K) para cobrir todos os dados até o final
        • eve-se indicar se os intervalos são abertos à esquerda ou a direita |-- ou --|
        • contar quantos dados incidem em cada classe
        • construir a tabela de frequências sem esquecer algum elemento (incidências, frequências relativas, densidades de frequências, linhas de totais)

Histograma

  • É uma representação gráfica da tabela de frequência dos dados
  • Não é um gráfico de barras das incidencias ni
  • Cada barra tem a largura de seu respectivo intervalo de classes deltai (no eixo-x)
  • Cada barra tem a altura da densidade de frequências di de seu respectivo intervalo de classes (no eixo-y)
  • Representa uma versão empírica (baseada nos dados amostrais) da distribuição de probabilidades da grandeza medida
  • O histograma corresponde a uma aproximação da distribuição de probabilidade da grandeza medida, por isso sua área (probabilidade) total vale ∑fiΔi = 1

Semana 3

Revisão de conteúdos básicos

  • Dígitos significativos
    • o número de dígitos significativos indica o erro máximo associado ao valor declarado
    • os limites numéricos dentro do qual se encontra o valor verdadeiro
    • e o nível de precisão associado ao valor. Exemplo: se x = 7 cm então o verdadeiro valor de x está compreendido em 6,5 e 7,5 cm. Se x = 7,0 cm então o verdadeiro valor de x está compreendido em 6,95 e 7,05 cm
  • Regra de arredondamento
    • o arredondamento não deve ser feito antes do final do cálculo. Deve ser feito ao final se o dígito seguinte ao dígito para o qual vamos arrendondar vale 0, 1, 2, 3 ou 4, truncamos o valor a partir daquele dígito. Exemplo: 3,14 → 3,1 (arredonda para baixo)
    • se for 6, 7, 8 ou 9 arredondamos para cima. Exemplo: 2,718 → 2,72
    • se o dígito for 5, o protocolo mais usado é arredondar para o par mais próximo. Exemplo: 3,1415 → 3,1412 / 0,5765 → 0,576 / 23,5 → 24. Sempre vai virar o par mais próximo
    • dentro dos computadores os números são sempre truncados
  • Progressôes e somas
    • Progressão aritmética
      • an = a0 + nr
      • Soma = (n + 1)(a1 + an)) / 2
    • Progressão geométrica
      • an = an-1.qn
      • Soma = a0(qn+1 - 1 / q - 1)
      • quando a razão da PG vale |q| < 1, a soma infinita vale a0/(1 - q)
    • Identidades que ocorrem com frequência
      • Soma de inteiros k de 1 até n = (n(n + 1)) / 2
